Cockpit Interaction System
1
"Lock" mode (default PC)
  • Left-click and hold the knob/switch
  • Push mouse forward → "push" interaction
  • Pull mouse back → "pull" interaction
  • Middle-mouse scroll or right-click "set" also available on some controls
Set under General Options → Accessibility → Cockpit Interaction System.
2
"Legacy" mode
  • An icon appears to the left/right/above/below the control
  • Circular arrow icon → middle-mouse wheel to scroll
  • Up/down arrow icon → left-click to set

Crew Hatch
1
Opening
  • Can only be opened while on the ground
  • Click the Open button on the cockpit overhead panel
  • Hatch and integrated ladder extend down to ground
2
Closing — required before taxi
  • Click the Close button on the same overhead panel
  • "Crew Hatch — CLOSED" is a required Before Start QRC item
Do not taxi with the hatch open.
Cargo Doors
1
Opening / Closing both cargo doors
  • Click the Cargo Door switch on the left instrument panel
  • Toggles both doors together — status shows OPEN or CLOSED
  • MFD STATS page shows READY FOR RELEASE once release criteria are met with doors open
Left console items: 4 = Cargo Release Button · 5 = Cargo Reload · 6 = Cargo Bay Doors Open/Close
Stealth Mode
1
What it does
  • Limits control-surface deflection animations
  • Hides navigation and beacon lights at wingtips
Visual/animation effect only — not a radar-signature simulation.
!
Activation requirements
  • Click Stealth Mode button on the left instrument panel
  • Must be above 3,000 ft AGL to activate
  • Set OFF before landing — required Landing QRC item
If the button has no effect, check altitude first — it cannot engage below 3,000 ft AGL.
