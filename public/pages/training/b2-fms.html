509th Bomb Wing · B-2 Spirit

FMS / CDU Keypad

COMM, NAV, PRFM, and FPL pages — every page on the Spirit's FMS keypad, plus the scratch-pad entry method used on all of them.

← Back to Training Center
COMM Page
1
COMM page
COM COMM1 [124.850] → AMASK T/R+G PLAIN COMM2 [124.850] → AMASK T/R+G PLAIN ADF [890.00] → USB PLAIN XPNDR [7000] → LINK RADIO SET
2
Scratch-pad entry — used on every FMS page
  • Type the new value on the keypad → appears in the scratch pad at the bottom of the screen
  • Press the line-select key beside the field to load the value
  • COMM1, COMM2, ADF, and XPNDR all use this method
Invalid entry shows INVALID INPUT. Press CLR to clear, then retype.
NAV Page
1
NAV page
NAV NAV1 [109.100] → --->U/A [U/A] NAV2 [0.000] → --->U/A [U/A] TACAN1 [1X] → TACAN2 [1X] → NAV MODE [GPS] →
  • VOR1/VOR2 and TACAN1/TACAN2 set using the same scratch-pad + line-select method
  • For ILS approach: NAV MODE must be set to NAV — not GPS
2
Valid frequency capture
NAV1 [110.300] → --->LOC/ILS [ISZL]
  • Once a valid frequency is captured, the ident and type display automatically
  • Confirm ident matches your approach plate
APP autopilot mode requires NAV MODE = NAV. If it's set to GPS, APP will not capture the ILS.
PRFM Page — Performance
1
PRFM page
PRFM V1 ZFW ← [136] [158.9] → VR ZFW CG ← [146] [30.7%] → V2 FOB ← [155] [70.1] → TEMP GW ← [13°C/56°F] [229.0] → GW CG [30.7%] →
  • V1, VR, V2 are automatically computed from weight and conditions — treat as a performance guide
  • ZFW / ZFW CG — Zero Fuel Weight and CG
  • FOB — Fuel On Board
  • GW / GW CG — Gross Weight and CG
FPL Page and DIR-TO
1
FPLN page
FPLN NO FPL ACTIVATED [----------]
  • Displays the active route from the World Map flight plan
  • Cannot be typed manually — must come from the sim
  • No valid flight plan shows NO FPL ACTIVATED
2
DIR — Direct To
DIR DIR TO: [CLPPR] DIST: [441NM] [ACTIVATE?] ←[YES] [CANCEL]→
  • Press line-select key next to any FPLN waypoint
  • Confirm with ACTIVATE? → YES
  • Press CANCEL to abort
