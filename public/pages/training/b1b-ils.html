ILS Approach

Step-by-step FMS, NAV page, and autopilot setup for a full ILS from vectors to touchdown or go-around. Example: KDYS ILS RWY 34L — 109.9 / I-DYS / CRS 359°.

← Back to Training Center
Phase 1 — FMS setup before the approach
1
FMS NAV page — tune the ILS
FMS → NAV NAV1 [109.9] → --->LOC/ILS [I-DYS] CRS [359] → NAV2 [0.000] → NAV MODE [NAV] →
  • Press NAV on the FMS keypad
  • Type 109.9 in the scratch pad → line-select into NAV1
  • Confirm ident shows LOC/ILS — I-DYS
  • Set CRS to 359° (inbound course for RWY 34L)
  • Confirm NAV MODE is NAV — not GPS
2
FMS PERF → APPR page — set DH
FMS → PERF → CRZ PG → APPR PG APPR PAGE FLAPS --> 17° VREF --> □□□ Kt ← auto-compute DH □□□□ RADIO ← enter DH from plate LAND QNH --> 29.92 inHg <--CRZ PG
  • Enter DH from approach plate (typically 200 ft HAT for CAT I)
  • Click VREF to auto-compute landing speed
  • Update QNH from current ATIS
Enter the Decision Height (DH) here, not MDA. ILS is a precision approach — you're looking for the runway at DH, not MDA.
Phase 2 — PFD softkey and APR mode
1
Set PFD softkey to ILS
VSD | ADF | ILS | NAV1 | GPS ↑ press this
  • The PFD nav mode softkey controls what the compass rose deviation needle tracks
  • For ILS: set to ILS — this shows the LOC needle and activates ILS bars in HUD mode
  • Wrong softkey → APR captures nothing
This is the most common student error. APR mode needs the PFD set to ILS — if it's on GPS or NAV1 the localizer will never capture.

Phase 4 — Decision Height and Go-Around
1
At Decision Height
  • At DH: look for runway environment
  • Runway in sight and landing assured → disengage AP, land normally
  • No runway contact at DH → immediate go-around
DH is a hard decision point. You must have the runway or approach lights in sight to continue below it. Never "peek" below DH hoping to find it.
!
Go-Around procedure
  • Disengage AP
  • Full power — positive rate — gear up
  • Climb runway heading to missed approach altitude
  • Published missed approach: climb to 3,000 ft, turn as published
  • Notify ATC: "KDYS Tower, [callsign], going around"
!
Common student errors
  • Wrong PFD softkey → LOC won't capture. Check softkey is ILS, not GPS.
  • NAV MODE set to GPS in FMS → APR won't work. NAV MODE must be NAV.
  • APR armed too early (more than 30° off final) → LOC may not sequence correctly. Arm APR when established within 30° of the inbound course.
  • Re-engaging HDG after LOC capture → instantly kills the LOC track. Once APR is green, leave HDG alone.
  • Forgetting to set DH in APPR page → no DH callout from the FMS PERF system.
