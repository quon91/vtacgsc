Common Student Errors — B-1B
  • ILS: Wrong PFD softkey (GPS instead of ILS). Coach: "Check your PFD softkey."
  • ILS: FMS NAV MODE left on GPS. Coach: "FMS NAV page — what's your NAV MODE?"
  • RNAV: Pressing APR instead of NAV HOLD. Coach: "NAV, not APR, for RNAV."
  • ATH: Expecting autothrottle to hold VREF below 2,500 ft AGL. Coach: "ATH is gone — take your throttle."
  • DH: Continuing past DH with no visual. Coach: "That was DH — go around now."
  • AR: Over-controlling in contact. Coach: "Small inputs. Stabilize first."
  • TOT: Late correction on run-in. Coach: "Check your timing now — before the IP."
📋
IP Pre-Brief Checklist — B-1B
  • Student states VR and VREF from FMS PERF before taxi
  • Student states active runway, inbound course, and ILS frequency before approach setup
  • Student states PFD softkey and A/P lateral mode for each approach type before brief ends
  • Go-around criteria explicitly briefed: IMC at DH, engine issue, ATC instruction
  • TOT window (±30 sec) and abort criteria briefed for all Phase 3 events
  • For AR: rejoin type, contact altitude, and breakaway triggers briefed before pushback

Common Student Errors — B-2
  • APP won't capture: FMS NAV MODE = GPS instead of NAV. First check every time.
  • Weapon release fails: Below 20,000 ft AGL or 230 KIAS. Have student call alt and speed aloud.
  • Stealth Mode won't engage: Below 3,000 ft AGL. Check altitude first.
  • Doors left open: Make door closure a verbal callout item on egress.
  • Version check: Confirm iniBuilds store version before any weapon delivery event.
📋
IP Pre-Brief Checklist — B-2
  • Confirm iniBuilds store version before any weapon delivery event
  • Student states 3 release criteria before stepping to aircraft
  • Student states FMS NAV MODE requirement for APP mode before brief ends
  • Brief Stealth Mode altitude restriction (3,000 ft AGL minimum)
  • Brief CFD flight model — handling differs from real Spirit FBW

Common Student Errors — B-52H
  • Phase 1: Snap rotation — coach steady, deliberate pitch-up, not fighter-style.
  • Phase 2 AR: Over-controlling at contact — small inputs, early breakaway call.
  • Phase 3 TOT: Drift uncorrected until IP — coach time checks throughout route.
  • Phase 3 Night: Lighting not reconfigured — make it a deliberate checklist item.
📋
IP Pre-Brief Checklist — B-52H
  • Student states TOLD V-speeds for current gross weight before taxi
  • For Phase 2: tanker rejoin type, contact altitude, and breakaway triggers briefed
  • For Phase 3: abort criteria (weather, terrain, TOT recovery limits) briefed before flight
  • Safe-escape maneuver and timing briefed before the IP — not first time on run-in
  • TOT window (±30 sec) confirmed understood and time-check plan briefed
