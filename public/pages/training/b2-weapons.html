Release Sequence
StepActionConfirm
1Climb above 20,000 ft AGL at 230+ KIASBoth conditions sustained
2Roll wings levelNo bank
3Check MFD STATS page for "READY FOR RELEASE"Message displayed
4Set Bomb Bay Doors switch → OPEN (left control panel)Door status reads OPEN
5Press WEAPON FIRE / Cargo Release buttonBombs observed releasing
6Set Bomb Bay Doors switch → CLOSEDDoor status reads CLOSED
After all bombs released: press and hold MAIN WPN REL to reload. Can be done at any point.
Instructor Notes
  • Version check first — confirm student has the iniBuilds store version before any troubleshooting. Microsoft Store/Xbox version does not include weapon delivery.
  • Most common error — release attempted below 20,000 ft AGL or below 230 KIAS. Have student call out altitude and airspeed before each attempt.
  • Door drag — close bomb bay doors promptly after release to avoid unnecessary drag on egress.
